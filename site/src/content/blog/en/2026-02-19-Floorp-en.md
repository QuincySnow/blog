Floorp is a Firefox-based browser with fine-grained tuning. This post covers performance (multi-core, WebRender, Stylo) and privacy (fingerprinting, tracking protection, memory settings).

## Overview

- **WebRender**: Rust-based GPU rendering
- **Stylo**: Rust-based CSS engine
- **Multi-process**: isolation and stability

Use `about:config` for advanced options; install add-ons from the Firefox (AMO) ecosystem. For step-by-step `about:config` and privacy tweaks, see the [Chinese version](2026-02-19-Floorp) of this article.
